Brief siap!Salin brief ini dan berikan ke asisten AI Agent pilihan kamu.
Perlu mengubah sesuatu?
Simpan ke Kanban BoardBrief akan masuk ke kolom Draft di Kalender Konten.

💡 Cara pakai brief ini

  1. Klik Copy Teks Brief untuk menyalin ringkasan siap pakai
  2. Paste ke asisten AI Agent kamu (Claude, ChatGPT, dsb.)
  3. Konten akan dibuat sesuai brief yang sudah kamu isi
  4. Gunakan Copy JSON untuk format terstruktur yang lebih mudah dibaca AI Agent
📋
Prompt ToR siap!Salin dan berikan ke AI Agent untuk membuat dokumen ToR lengkap.
Perlu mengubah data kegiatan?
Simpan ke Kanban BoardBrief akan masuk ke kolom Draft di Kalender Konten.

💡 Cara pakai prompt ToR ini

  1. Klik Copy Prompt ToR di atas
  2. Buka AI Agent pilihan kamu (Claude, ChatGPT, dsb.)
  3. Paste dan kirim — AI Agent akan membuat dokumen ToR lengkap
  4. Dokumen sudah mencakup judul, latar belakang, tujuan, dan hasil yang diharapkan

💡 Cara pakai prompt ini

  1. Klik Copy Prompt di atas
  2. Buka AI Agent (Claude, ChatGPT, dsb.)
  3. Paste prompt — AI akan membuat konten sesuai format yang kamu pilih
  4. Jika hasilnya belum sempurna, minta AI untuk merevisi bagian tertentu
